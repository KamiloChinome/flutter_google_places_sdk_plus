# google_places_sdk_plus_linux

[![pub package](https://img.shields.io/pub/v/google_places_sdk_plus_linux.svg)](https://pub.dev/packages/google_places_sdk_plus_linux)

The Linux implementation of [`google_places_sdk_plus`](https://pub.dev/packages/google_places_sdk_plus).

Delegates to [`google_places_sdk_plus_http`](https://pub.dev/packages/google_places_sdk_plus_http) for HTTP/REST access.

## Usage

This package is endorsed and automatically included when you depend on `google_places_sdk_plus`. You do not need to add it to your `pubspec.yaml` directly.

## Restrictions

The following methods are not yet implemented:

- `fetchPlace`
- `fetchPlacePhoto`
