# google_places_sdk_plus_ios

[![pub package](https://img.shields.io/pub/v/google_places_sdk_plus_ios.svg)](https://pub.dev/packages/google_places_sdk_plus_ios)

The iOS implementation of [`google_places_sdk_plus`](https://pub.dev/packages/google_places_sdk_plus).

Uses the [Google Places SDK for iOS](https://developers.google.com/maps/documentation/places/ios-sdk) (New) via Swift.

## Usage

This package is endorsed and automatically included when you depend on `google_places_sdk_plus`. You do not need to add it to your `pubspec.yaml` directly.
