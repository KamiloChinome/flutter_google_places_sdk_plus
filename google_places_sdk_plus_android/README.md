# google_places_sdk_plus_android

[![pub package](https://img.shields.io/pub/v/google_places_sdk_plus_android.svg)](https://pub.dev/packages/google_places_sdk_plus_android)

The Android implementation of [`google_places_sdk_plus`](https://pub.dev/packages/google_places_sdk_plus).

Uses the [Google Places SDK for Android](https://developers.google.com/maps/documentation/places/android-sdk) (New) via Kotlin.

## Usage

This package is endorsed and automatically included when you depend on `google_places_sdk_plus`. You do not need to add it to your `pubspec.yaml` directly.
